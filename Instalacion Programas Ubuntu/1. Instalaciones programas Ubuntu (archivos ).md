
En Ubuntu, los programas pueden instalarse de distintas formas según su origen y el método de distribución. Principalmente se dividen en **instalación mediante archivos** y **gestores de paquetes**.

---

## 1. Instalación mediante archivos

Este método consiste en instalar software directamente desde archivos descargados, sin usar un gestor de paquetes centralizado.

### 1.1 Archivos `.deb`

Son paquetes propios de Debian y Ubuntu.

**Instalación por terminal:**

```bash
sudo dpkg -i programa.deb
```

Si hay errores de dependencias:

```bash
sudo apt --fix-broken install
```

**Instalación gráfica:**

- Haz doble clic en el archivo `.deb` para abrirlo con el instalador de paquetes.
    

---

### 1.2 Archivos AppImage

Aplicaciones empaquetadas en un único archivo ejecutable.

**Pasos para usarlo:**

```bash
chmod +x programa.AppImage
./programa.AppImage
```

- No requieren instalación en el sistema.
    
- Son portables y fáciles de ejecutar.
    

---

### 1.3 Archivos `.bundle`

Instaladores empaquetados que suelen contener binarios y scripts de instalación.

**Instalación:**

```bash
chmod +x programa.bundle
sudo ./programa.bundle
```

- Generalmente requieren permisos de administrador.
    
- Suelen usarse para software propietario o de terceros.
