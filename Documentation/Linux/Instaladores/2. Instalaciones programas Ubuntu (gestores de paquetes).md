## 2. Gestores de paquetes

Recomendados para instalar software de manera segura, con gestión de dependencias y actualizaciones.

### 2.1 APT (Advanced Package Tool)

```bash
sudo apt update
sudo apt install nombre-paquete
sudo apt remove nombre-paquete
sudo apt upgrade
```

### 2.2 Snap

```bash
sudo snap install nombre-paquete
```

- Incluye dependencias.
    
- Actualizaciones automáticas.
    

### 2.3 Flatpak

```bash
sudo apt install flatpak
flatpak install flathub nombre-aplicacion
```

- Buen aislamiento.
    
- Ideal para aplicaciones gráficas.
