# Guía de Instalación de Programas en Ubuntu - Índice

Esta nota sirve como antesala para explorar los distintos métodos de instalación de software en Ubuntu. Desde aquí puedes navegar fácilmente a cada sección detallada en sus respectivas notas usando los enlaces de Obsidian.

## Enlaces a notas detalladas

- [[1. Instalaciones programas Ubuntu (archivos )|Instalación mediante archivos]]  
    Explicación de cómo instalar programas usando archivos descargados, incluyendo `.deb`, AppImage y `.bundle`.
    
- [[2. Instalaciones programas Ubuntu (gestores de paquetes)|Gestores de paquetes]]  
    Guía sobre cómo instalar software usando gestores de paquetes como APT, Snap y Flatpak, con ventajas y comparativa.
    
- [[3. Comparativa rápida|Comparativa de métodos de instalación]]  
    Tabla rápida para comparar dependencias, actualizaciones e integración de cada método.
    
- [[4. Recomendaciones|Recomendaciones de instalación]]  
    Consejos sobre qué método usar según el tipo de software y necesidades del usuario.
    

Esta nota funciona como un **punto de entrada** para explorar los detalles de cada método de instalación de manera organizada y navegable dentro de Obsidian.
