# Guía Rápida: Cómo Escribir en Markdown

Markdown es un lenguaje de marcado ligero que permite dar formato a un texto de manera sencilla y rápida. Aquí tienes los comandos esenciales:

---

## 1. Títulos
Se crean usando el símbolo `#`.
# Título Nivel 1
## Título Nivel 2
### Título Nivel 3

## 2. Formato de Texto
* **Negrita:** `**texto**`
* *Cursiva:* `*texto*`
* ***Negrita y Cursiva:*** `***texto***`
* ~~Tachado:~~ `~~texto~~`

## 3. Listas
**Desordenada:**
* Elemento A
* Elemento B
  * Subelemento

**Ordenada:**
1. Primero
2. Segundo

## 4. Enlaces e Imágenes
* **Enlace:** `[Texto del enlace](https://www.ejemplo.com)`
* **Imagen:** `![Descripción](https://url-de-la-imagen.jpg)`
